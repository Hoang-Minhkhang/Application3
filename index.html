Chào mừng đến với Application3

Bấm nút bên dưới để tải và cài đặt ứng dụng:

Tải ứng dụng (ClickOnce)

Tải file Setup trực tiếp

Lưu ý: Đây là ứng dụng ClickOnce. Sau khi tải file .application, hãy mở file đó để bắt đầu quá trình cài đặt.
Nếu gặp lỗi, bạn có thể tải file setup.exe để cài đặt trực tiếp.
